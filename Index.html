About Joan

I’m Joan — a passionate and creative Frontend Engineer dedicated to building beautiful, responsive, and user-friendly websites. I specialize in turning designs into interactive, pixel-perfect digital experiences using HTML, CSS, JavaScript, and modern frameworks like React

My approach to frontend development blends technical precision with a strong eye for design. I believe every interface should not only function flawlessly but also feel intuitive and visually engaging. Whether I’m creating a personal portfolio, a brand website, or a user dashboard, I focus on performance, accessibility, and elegant UI

Beyond the code, I love exploring the world of design trends, experimenting with animations, and continuously learning new tools to stay ahead in this fast-paced tech space. For me, frontend development is more than a career — it’s a creative outlet where logic meets beauty.

Let’s build something beautiful, functional, and unforgettable.

Design First

I believe in starting with a solid design foundation before diving into code. Understanding the visual hierarchy and user flow is essential for creating effective interfaces.

Mobile-First Approach

With the increasing use of mobile devices, I prioritize mobile-first design to ensure optimal user experience across all screen sizes.

Accessibility Matters

I'm committed to creating websites that are accessible to all users, regardless of their abilities or disabilities.

Clean Code

I strive to write clean, maintainable code that is easy to understand and build upon, following best practices and industry standards.
